## Description

E-commerce fullstack applicaiton. Live demo on heroku [https://lignum-furniture.herokuapp.com/](https://lignum-furniture.herokuapp.com/)

## Installation

```bash
$ npm install
```

## Running the app

```bash

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Open [http://localhost:8000](http://localhost:8000) to view it in your browser.
